R Short Course

W. Andrew Barr

Welcome to this short course in the R statistical programming language. The plan is to get you doing basic data manipulation, graphing, and analysis as quickly as possible. We won’t cover basic statistical concepts, but rather we will focus on how to implement the analyses that you already know to do in R.

R is the most popular and most flexible tool for doing scripted data analysis currently available. There is a rather steep learning curve in R, but once you get ‘over-the-hump’ it will make your data analysis easier, faster, and more reproducible.

Let’s get started with the task of getting data in and out.
